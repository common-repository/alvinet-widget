Lecteur de flux RSS avancé utilisant l'API d'Alvinet. Affiche toute l'actualité francophone sur votre site : Politique, économie, faits divers, multimédia, culture... Sélectionnez simplement l'actu qui vous intéresse.

== Description ==

Affichez, sur votre site, le widget actualité d'[Alvinet](http://www.alvinet.com) et offrez ainsi, à vos utilisateurs, des sources d'informations en rapport direct avec le thème abordé dans vos pages.

[Alvinet](http://www.alvinet.com) référence chaque jour dans sa base de données des milliers d'articles de presse en provenance des principaux médias francophones. Pour compléter vos articles, vos pages web, votre blog, il peut parfois être utile de faire référence à des sources extérieures directement en rapport avec le sujet que vous traiter. Ces sources, au travers de son API, [Alvinet](http://www.alvinet.com) vous les sert sur un plateau, toujours actualisées. Les articles présentés sont les plus récents sur le thème que vous aurez choisis.

== Installation ==

Décompressez l'archive et placez le dossier `alvinet` dans le répertoire `/wp-content/plugins/` de votre installation WordPress. Activez le plugin depuis le menu `Extensions` de Wordpress.
